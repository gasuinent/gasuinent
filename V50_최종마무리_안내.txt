가수인생활 홈페이지 V50 최종 자료복구 통합본

포함 자료
- 뉴스 6건
- 공지사항 4건
- 행사일정 3건
- 주요 실적 4건
- 갤러리 사진 3장
- 관리자 폴더(admin)
- 데이터 폴더(data)
- 업로드 폴더(uploads)
- 실제 다운로드 가능한 자료실(resources_files)

보완 내용
1. data 파일의 일부 자료가 누락되어도 기준 자료와 자동 병합합니다.
2. 갤러리는 기준 사진 3장을 유지합니다.
3. 뉴스·공지·일정·실적의 기준 자료가 빠지지 않도록 보호했습니다.
4. 자료실에서 등록자료 목록을 직접 내려받을 수 있습니다.

적용 방법
압축을 풀고 내부의 모든 파일과 폴더를 GitHub 저장소 맨 위에 업로드합니다.
ZIP 파일 자체는 올리지 마십시오.
