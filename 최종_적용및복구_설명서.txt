가수인생활 엔터테인먼트 홈페이지 V49 최종 운영·복구 통합본

1. 이 압축파일의 용도
- 현재 홈페이지에 적용하는 최종 운영본
- 오류 발생 시 되돌리는 복구용 원본
- 관리자, 뉴스, 공지, 일정, 실적, 갤러리 자료를 모두 포함

2. 중요한 폴더
- admin : 관리자 페이지
- data : 뉴스·공지·일정·실적·갤러리 등록자료
- uploads : 관리자에서 올린 이미지
- 저장소 맨 위 index.html : 방문자용 홈페이지

3. GitHub 적용 방법
- 압축을 풉니다.
- 압축을 푼 폴더 안의 파일과 폴더 전체를 GitHub 저장소 main의 맨 위(root)에 올립니다.
- 같은 이름의 파일은 교체합니다.
- admin, data, uploads 폴더를 빠뜨리지 않습니다.
- Commit changes를 누릅니다.

4. 적용 후 확인
- 공개 홈페이지: https://gasuinent.netlify.app/
- 관리자: https://gasuinent.netlify.app/admin/
- 새로고침: Ctrl + F5
- 뉴스에서 카테고리별 기사 표시 확인
- 공지 펼치기 확인
- 일정·실적 세부보기 확인
- 갤러리 사진 3장 확인

5. 복구 방법
- 문제가 생기면 이 압축파일을 다시 풀어 전체 파일을 GitHub root에 덮어씁니다.
- 파일검증_SHA256.txt로 원본 손상 여부를 확인할 수 있습니다.

주의
- 저장소 맨 위 index.html과 admin/index.html은 서로 다른 파일입니다.
- 관리자 주소를 명함이나 홍보물에 공개하지 않습니다.
